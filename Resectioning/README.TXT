Usage:

Examples of usage of the toolbox are in the files example_tri.m and example_res.m which describe how to do triangulation and resectioning respectively. You will need a recent version of SeDuMi to use this toolbox. SeDuMi can be downloaded freely from http://sedumi.mcmaster.edu/.
